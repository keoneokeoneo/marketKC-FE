# Market KC Front End

캡스톤 디자인2 - 스마트 컨트랙트를 이용한 중고거래 시스템의 프론트엔드 프로젝트입니다.

### 사용된 기술 스택

- [React-Native](https://reactnative.dev/)
- [Typescript](https://www.typescriptlang.org/)
- [Redux](https://redux.js.org/)
- [Redux-Thunk](https://github.com/reduxjs/redux-thunk)
- [React-Navigation V5]
- [React-Hook-Form]
